# 🎯 Number Guessing Game (Python)

## 📌 Project Description

The **Number Guessing Game** is a simple Python-based console game where the computer randomly selects a number, and the player tries to guess it within a limited number of attempts.

This project is designed to demonstrate basic programming concepts such as loops, conditionals, user input handling, and random number generation.

---

## 🚀 Features

* Random number generation
* User-friendly input system
* Hints like **"Too High"** or **"Too Low"**
* Limited number of attempts
* Replay option (optional)

---

## 🛠️ Technologies Used

* Python 3
* Built-in modules:

  * `random`

## ▶️ How to Run the Project

1. Make sure Python is installed on your system
2. Clone the repository or download the files
3. Open terminal in project folder
4. Run the program:

```
python main.py
```

---

## 🎮 How to Play

1. The computer generates a random number (e.g., between 1 and 100)
2. You enter your guess
3. The program will respond:

   * 📉 Too Low
   * 📈 Too High
   * ✅ Correct Guess
4. Continue until you guess correctly or run out of attempts

---

## 💡 Example Output

```
Enter a number between 1 and 100: 50
Too High!

Enter a number: 25
Too Low!

Enter a number: 30
Correct! You guessed the number 🎉
```
